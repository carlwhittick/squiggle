squiggle

Canvas-drawn animated underlines that ripple outward from hover and click, with skip-ink around glyphs.

GitHub npm

Live demo

Hover or click these rippling words to see the wave propagate.

Squiggle replaces native text-decoration with an absolutely-positioned canvas that draws a damped travelling wave along the baseline. It reads the element's computed color, thickness, and underline offset — so it respects CSS decoration properties out of the box.

Each mouseenter or click plants a ripple origin. Multiple overlapping ripples superpose correctly, and a ResizeObserver keeps the canvas in sync as layout changes.

Works at any size — small text links get the same skip-ink treatment, gapping around descenders and ascenders so the line never punches through a g, j, p, q, y.

Usage

// install
npm install @carlwhittick/squiggle

// import
import { squiggleUnderline } from '@carlwhittick/squiggle';

// attach to any CSS selector
const instance = squiggleUnderline('a, .link');

// with options
const instance = squiggleUnderline('a', {
  amplitudeEm:      0.2,   // wave half-height as em multiplier
  frequency:        4,     // Hz
  propagationSpeed: 300,   // px / s
  timeDamping:      2.5,   // amplitude decay over time
});

// clean up (SPA route change, etc.)
instance.destroy();

Options

Option Default Description
amplitudeEm 0.2 Wave half-amplitude as em multiplier
wavelengthBase 16 Base wavelength in px
wavelengthEm 0.5 Wavelength em contribution
frequency 4 Oscillation frequency (Hz)
propagationSpeed300 Ripple travel speed (px/s)
spatialDamping 0.012 Amplitude decay with distance
timeDamping 2.5 Amplitude decay over time
thicknessEm 0.075 Stroke thickness as em multiplier
offsetYEm -0.05 Vertical offset from baseline
skipInkGapEm 0.1 Gap around glyphs (skip-ink)
